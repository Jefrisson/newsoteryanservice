# BreachShield · Data Intelligence

Универсальный BI-инструмент над JSON: загружаешь любой файл — получаешь дашборд с автодетекцией схемы, фильтрами, графиками и поиском. Всё локально, без бэкенда.

## Что умеет

- **Auto-schema** — сканирует JSON и определяет тип каждого поля: number / string / category / date / boolean / object / array
- **Multi-dataset** — если в JSON несколько массивов (пример: `{users:[...], orders:[...]}`), переключается между ними одним кликом
- **Smart filters** — для каждого поля свой контрол: multi-select для категорий, range для чисел, date-pickers для дат, contains для текста, true/false для boolean
- **Drill-down** — клик по любому значению на графике или в bar-chart карточки → фильтр применяется ко всему дашборду
- **3 представления**:
  - **Overview** — авто-дашборд с карточкой на каждое поле (мини-гистограмма для чисел, top-list для категорий, sparkline для дат, pie для boolean)
  - **Explore** — таблица с сортировкой, выбором колонок, пагинацией, deep-view карточкой записи
  - **Reports** — конструктор графиков: выбираешь chart type / X / Y / aggregation / group-by → живой превью
- **Глобальный поиск** (`Ctrl/Cmd + K`) — full-text по всем полям всех записей
- **CSV-экспорт** — выгружает отфильтрованные данные одной кнопкой
- **Drag & drop** — перетащил `.json` в окно браузера → дашборд готов

## Как запустить

1. Открой `index.html` локально или хости любым способом
2. По умолчанию подхватит `data.json` из той же папки (в репо лежит пример с CRM-данными)
3. Кнопка **Load JSON file** в сайдбаре или drag-drop — для загрузки своего
